  {implementationTitle} [опции]

Опции
  -h               Выводит эту справку
  --help

  -i               Интерактивная работа
  --interactive

  --skipHello      Не выводить в начале интерактивного действия 
                   приветствие

  --userInitScripts=true
                   Запускать автоматические скрипты

  --charsets       Выводит список кодировок

  --cs=Кодировка   Указывает кодировку, как для файла, так и для выводимых
                   данных.
                   Значение default - устанавливает кодировку по умолчанию.

  --endl=Перевод   Указывает символы перевода строки, как для выводимых данных,
                   так и для лога.
                   Возможные значения:
                     windows
                     linux
                     mac
                     other
                     default

  -l лог_файл      Пишет в лог-файл выполняемый исходный код,
  --log=лог_файл   результат выполняния и ошибки.

  -f Файл          Файл для исполнения
  --file=Файл

  --exp=Выражение  Вырожение которое необходимо исполнить
  -e Выражение
